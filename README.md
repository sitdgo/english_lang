# Telegram-бот «EnglishCard»

**Курсовая работа по дисциплине «Базы данных»**

Я разработал Telegram-бота для изучения английского языка. У каждого пользователя своя отдельная база слов, бот следит за прогрессом и помогает учить новые слова через тесты.

## Что умеет бот

- **Свой словарь у каждого пользователя** — всё персонально, ничего не перемешивается.
- **Стартовый набор** — при первом запуске бот сразу добавляет 10 самых нужных базовых слов.
- **Тестирование** — показывает английское слово и 4 варианта перевода, нужно выбрать правильный.
- **Добавление слов** — можно в любой момент добавить новое слово (английское + перевод).
- **Удаление слов** — если слово уже выучено или не нужно, его легко убрать.

## Технологии, которые я использовал

- Язык программирования: **Python 3**
- База данных: **PostgreSQL**
- Библиотеки:
  - `pyTelegramBotAPI` — для работы с Telegram
  - `psycopg2-binary` — для подключения к PostgreSQL

## Структура базы данных

Я сделал три таблицы, связанные отношением «многие ко многим»:

1. **users** — хранит chat_id пользователей (чтобы каждый имел свои слова).
2. **words** — все английские слова и их русские переводы.
3. **user_words** — связующая таблица, показывает, какие слова сейчас изучает конкретный пользователь.

Схема базы данных лежит в файле `schema.drawio.png` в корне проекта.
